# DSA210_TermProject
# DSA210 Term Project

## Project Title
The Championship DNA: Analyzing Differentiators Between League Champions and Runner-Up Teams in European Football

## Project Overview
This project investigates whether league champions differ systematically from runner-up teams across major European football leagues.

For the April 14 checkpoint, the project focuses on:
- data collection
- exploratory data analysis (EDA)
- hypothesis testing

The broader project proposal also includes financial, tactical, and temporal variables. At this stage, the goal is to build a reliable cross-league dataset and test the first set of hypotheses.

## Research Questions / Hypotheses
1. Champions and runner-up teams do not differ significantly in total wage bill.
2. Champions do not use a significantly wider rotation than runner-up teams.
3. Champions use the 4-3-3 formation in a larger share of matches than runner-up teams.
4. Champions have a higher average squad age than runner-up teams.

## Planned Variables
- total wage bill
- rotation size
- 4-3-3 usage rate
- average squad age
- league
- season
- rank (1st or 2nd)
